# SpilCaf-en---Projekt-IxD

I dette projekt med fokus på IxD, har vi i vores gruppe, lavet en hjemmeside i mobil størrelse til bratspilcafeén: spilcaféen.
Projektet er lavet af, Celina, Clara, Nanna & Benjamin. 
Dette element af opgaven består af den kodet prototype, som er interaktiv. 
Den er lavet ved brug af HTML, CSS og JS - hvoraf alt data er hentet fra den givende JSon fil, som er blevet udleveret. 
Den er meget konpleks og der er meget kode. Så er du advaret.
